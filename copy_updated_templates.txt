To update the template files in your project, you need to copy the following files from the repository to your project directory:

1. Copy the updated edit_css.tt file:
   cp /comserv/Comserv/root/admin/theme/edit_css.tt.final /home/shanta/PycharmProjects/comserv2/Comserv/root/admin/theme/edit_css.tt

2. Copy the updated Theme.pm controller:
   cp /comserv/Comserv/lib/Comserv/Controller/Admin/Theme.pm /home/shanta/PycharmProjects/comserv2/Comserv/lib/Comserv/Controller/Admin/Theme.pm

3. Copy the updated ThemeConfig.pm model:
   cp /comserv/Comserv/lib/Comserv/Model/ThemeConfig.pm /home/shanta/PycharmProjects/comserv2/Comserv/lib/Comserv/Model/ThemeConfig.pm

These commands will copy the updated template files to your project directory, replacing the old versions with new versions that properly display all attributes from the theme_definitions.json file.

After copying these files, your application should:
1. Correctly display all theme attributes from the theme_definitions.json file in a single table
2. Allow you to browse and select images from the image directories
3. Properly save changes to the CSS file
4. Update the theme_definitions.json file with any CSS variables found in the CSS content
