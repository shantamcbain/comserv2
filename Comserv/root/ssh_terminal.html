⚠️ IMPORTANT: This is a placeholder for the SSH terminal interface.

📋 To enable interactive SSH terminal:

1. Install wetty or ttyd on workstation.local

2. Run: wetty --port 3001 --host 0.0.0.0

3. Or use: ttyd -p 3001 ssh ubuntu@192.168.1.126

Current Workaround: Use the commands below and paste into your local terminal

METHOD 1: Open Local Terminal

Open your PyCharm terminal or local terminal and run:

ssh ubuntu@192.168.1.126

METHOD 2: PyCharm Built-in SSH

1. PyCharm → Tools → Start SSH Session

2. Enter: ubuntu@192.168.1.126

3. Enter password when prompted

METHOD 3: Setup Web-Based Terminal (One-time setup)

On workstation.local, run:

npm install -g wetty wetty --port 3001 --host 0.0.0.0 --ssh-host 192.168.1.126 --ssh-user ubuntu

Then refresh this page
