# Todo Workflow

Every task worked on via the Comserv application todo system MUST follow this workflow in order:

1. **Open the todo** in the application — review the full description before starting
2. **Create a log entry** from the todo using the "Add Log" link on the todo detail page (this pre-fills the log form from the todo)
3. **Do the work**
4. **Close the todo** — set status to DONE
5. **Close the log** — the application automatically records the time spent; no manual time entry needed

Do NOT skip creating the log before starting work. Do NOT manually record time in the log — the software handles that on close.
