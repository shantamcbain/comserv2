# Template Usage Standards

Use Documentation/ApplicationTtTemplate.tt as the template base for all application.tt files outside the Documentation directory. Use Documentation/DocumentationTtTemplate.tt as the base for all documentation files inside the Documentation directory. When creating or editing TT files, adapt from these templates to maintain consistency.
