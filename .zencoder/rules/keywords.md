## IMMEDIATE EXECUTION PROTOCOL

**When You See a Keyword Command**:
1. **STOP analyzing** - Keyword commands are direct orders, not requests for analysis
2. **IMMEDIATELY execute** without preamble or explanation
3. **Reference the correct file**: keywords-global.md or keywords-disabled-tasks.md
4. **Do NOT ask questions** unless the keyword definition itself requires input

**Command Formats That Trigger Execution**:
- Slash format: `/chathandoff`, `/sessionhandoff`, `/updateprompt`, `/validatett`
- Legacy format: `Execute Keyword: Chat Handoff`, etc.

**Protocol Violation Handling**:
- If user says: `PROTOCOL VIOLATION - EXECUTE NOW` → **EXECUTE IMMEDIATELY** with all original parameters
- No further discussion allowed

**Hybrid Scenario**:
- If both `/chathandoff` AND `/sessionhandoff` appear: **ONLY execute /sessionhandoff** (it makes chathandoff redundant)
